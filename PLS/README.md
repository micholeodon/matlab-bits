# PLS

Scripts testing examples providedn in the paper "Partial Least Squares (PLS) Methods for Neuroimaging: A Tutorial and Review", Krishnan et al. 2011

## Usage

To check the example aimed to show how one can calculate correlation between PET scan data and behavioral measures run:

```
>> PLSC_Behavior.m
```
